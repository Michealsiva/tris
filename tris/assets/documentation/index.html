    Javascript

    It's possible to change the JS settings by extending the defaults from the template, that way is so much easier to update the theme changing only one JS file. Here are a few samples on how to do that:

    Change Settings

    Changing Scroll-to-Top Icon : (js/custom.js) 

     $('.scroll-to-top').find($(".fa")).removeClass('fa-chevron-up').addClass('fa-chevron-up');

    Find the above line in custom.js file and change the font-awesome icon class in addClass.
